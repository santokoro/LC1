Текстовый редактор «Compiler»
Назначение
Приложение представляет собой текстовый редактор на базе WPF (C#). Оно позволяет создавать, открывать, редактировать и сохранять текстовые файлы, 
а также отображать введённый текст в отдельной области вывода. Интерфейс включает меню, панель инструментов, нумерацию строк, строку состояния и поддержку горячих клавиш.

Возможности:
1.Работа с текстом
2.Редактирование текста в основном поле ввода.
3.Нумерация строк.
4.Поддержка переноса строк, табуляции и прокрутки.
5.Операции копирования, вырезания и вставки.
6.Undo/Redo на основе стеков.

Работа с файлами:
1.Создание нового документа.
2.Открытие существующих файлов.
3.Сохранение и «Сохранить как…».
4.Предупреждение о несохранённых изменениях.
5.Открытие файла перетаскиванием в окно (Drag & Drop).

Область вывода:
1.Отображение текста из редактора по нажатию кнопки «Пуск».
2.Возможность расширения под вывод результатов обработки.

Управление шрифтом:
1.Изменение размера шрифта кнопками.
2.Ввод размера вручную.

Строка состояния
Отображает:
1.Текущую строку и столбец курсора;
2.Количество строк;
3.Имя файла;
4.Статус документа («Сохранён» / «Изменён»);
5.Размер шрифта.

Горячие клавиши
Ctrl + N — новый файл

Ctrl + O — открыть файл

Ctrl + S — сохранить

Ctrl + Shift + S — сохранить как

Ctrl + Q — выход

Ctrl + Z — отмена

Ctrl + Y — повтор

Ctrl + C — копировать

Ctrl + X — вырезать

Ctrl + V — вставить

Ctrl + R — выполнить команду «Пуск»

Структура интерфейса
Поле ввода — основная область редактирования.

Поле вывода — отображение результата работы «Пуск».

Меню — команды работы с файлами и текстом.

Панель инструментов — быстрый доступ к основным действиям.

Строка состояния — информация о текущем состоянии документа.

Технические детали
Язык: C#

Технология: WPF (.NET)

Реализация Undo/Redo: два стека состояний

Загрузка справки и информации «О программе»: HTML-файлы через WebBrowser

Drag & Drop: обработка событий PreviewDragEnter и PreviewDrop

Рекомендации по использованию
Сохраняйте документ перед открытием нового файла.

Используйте горячие клавиши для ускорения работы.

Следите за строкой состояния для контроля позиции курсора и статуса файла.
