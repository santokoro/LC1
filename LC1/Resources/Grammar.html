Грамматика

Определим грамматику объявления и инициализации вещественной константы в языке Kotlin G[<F>] в нотации Хомского с продукциями P:
        <F> -> 'const' <CONST> | 'ε' <SPACE>
        <CONST> -> ' ' <SPACE>
        <SPACE> -> 'val' <VALUE>
        <VALUE> -> ' ' <SPACEAFTER>
        <SPACEAFTER> -> letter <ID>
        <ID> -> letter <ID> | digit <ID> | '_' <ID> | ':'<AFTER_COLON>
        <AFTER_COLON> -> 'Double' <TYPE>
        <TYPE> -> '=' <EQUAL>
        <EQUAL> = '-' <INT> | digit <INTREM> |
        <INT> -> digit <INTREM>
        <DIGIT > -> '0'|'...'|'9'
        <INTREM> -> digit <INTREM> | '.' <DOUBLE> |';' <END>

Следуя введенному формальному определению грамматики, представим G[⟨F⟩] ее составляющими:

− Z = < F >

− V_T = {a…z,A…Z,0…9,=,_,;,-.,}

− V_N = {<F>, <CONST>, <SPACE>, <VALUE>, < SPACEAFTER>, <ID>, <AFTER_COLON>, <TYPE>, <EQUAL>, <INT>, <INTREM>, <DIGIT>, <DOUBLE>, <END>}
