Доп. задание ЛР7 — блок-схемы локальных оптимизаций

Конструкция: const val id: Double = <выражение>. Откройте файл в браузере → скриншот для отчёта (Печать → PDF тоже подойдёт).

Оптимизация №1: Свёртка констант

flowchart TD
    A([Начало]) --> B[На вход: шаги вычисления выражения в промежуточном представлении]
    B --> C[Определить тип константы Double]
    C --> D[Создать список известных числовых значений промежуточных результатов]
    D --> E[Создать новый список шагов — результат оптимизации]
    E --> F{Есть ещё шаги в исходном списке?}
    F -->|Нет| Y[Пронумеровать шаги в новом списке]
    Y --> Z([Конец])

    F -->|Да| G[Взять следующий шаг]
    G --> H{Шаг — объявление константы?}
    H -->|Да| I[Перенести шаг в новый список без изменений]
    I --> F

    H -->|Нет| J{Шаг — запись числового литерала в промежуточный результат?}
    J -->|Да| K[Запомнить число для этого результата]
    K --> L[Записать шаг с литералом в каноническом виде для Double]
    L --> F

    J -->|Нет| M{Значения операндов уже известны?}
    M -->|Нет| N[Скопировать шаг в новый список без изменений]
    N --> F

    M -->|Да| O{Применимо алгебраическое упрощение?}
    O -->|Да| P[Получить результат: умножение на 0, сложение с 0, вычитание 0, умножение на 1]
    O -->|Нет| Q[Вычислить операцию: сложение, вычитание, умножение, деление или унарный минус]
    P --> R[Запомнить число для промежуточного результата]
    Q --> R
    R --> S[Заменить операцию одним шагом «известное числовое значение»]
    S --> F

Оптимизация №2: Устранение мёртвого кода и временных

flowchart TD
    A([Начало]) --> B[На вход: представление после свёртки констант]
    B --> C[Найти шаг объявления константы с итогом выражения]
    C --> D{Объявление найдено и итог привязан к результату выражения?}
    D -->|Нет| E[Вернуть вход без изменений]
    E --> F([Конец])

    D -->|Да| G[Создать список известных значений промежуточных результатов]
    G --> H{Есть ещё шаги до объявления?}
    H -->|Нет| M{Значение итогового результата выражения найдено?}
    H -->|Да| I[Взять следующий шаг]
    I --> J{Шаг записывает литерал в промежуточный результат?}
    J -->|Да| K[Запомнить: результат равен этому числу]
    K --> H
    J -->|Нет| L{Шаг — операция и значения операндов известны?}
    L -->|Да| N[Вычислить и запомнить значение результата]
    N --> H
    L -->|Нет| H

    M -->|Нет| E
    M -->|Да| O[Взять итоговое число выражения]
    O --> P[Сформировать новое представление из одного шага: объявить константу с готовым числом]
    P --> Q[Пронумеровать шаг]
    Q --> F
