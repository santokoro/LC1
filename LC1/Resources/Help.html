Справка по программе

Данное приложение представляет собой простой текстовый редактор, предназначенный для работы с текстовыми файлами и отображения введённого текста в отдельной области вывода.

Интерфейс программы

Основные области

  • Поле ввода — основное текстовое поле, в котором пользователь набирает и редактирует текст.
  • Поле вывода — область, в которой отображается результат работы кнопки «Пуск» (введённый текст).
  • Строка заголовка — отображает название программы и имя текущего файла.
  • Меню и панель инструментов — содержат команды работы с файлами, текстом и запуском обработки.

Работа с файлами

Создание нового файла

Команда «Новый» очищает поле ввода и вывода. Если в текущем документе есть несохранённые изменения, программа предложит сохранить их.

Открытие файла

Команда «Открыть» позволяет выбрать существующий текстовый файл. Если в текущем документе есть несохранённые изменения, перед открытием нового файла будет предложено их сохранить.

Сохранение файла

  • «Сохранить» — сохраняет изменения в текущий файл. Если файл ещё не был сохранён, будет предложено указать имя и место сохранения.
  • «Сохранить как…» — всегда предлагает выбрать имя файла и папку для сохранения.

Редактирование текста

Базовые операции

  • Копировать — копирует выделенный текст в буфер обмена.
  • Вырезать — копирует выделенный текст в буфер обмена и удаляет его из документа.
  • Вставить — вставляет текст из буфера обмена в позицию курсора.

Отмена и повтор действий

  • Назад — отменяет последнее изменение текста.
  • Вперёд — повторяет отменённое действие, если это возможно.

История изменений сохраняется только в рамках текущего сеанса работы с документом.

Кнопка «Пуск»

Кнопка «Пуск» берёт текст из поля ввода и отображает его в поле вывода без изменений. Таким образом, пользователь может увидеть введённый текст во второй области окна.

Горячие клавиши

Управление файлами

  • Ctrl + N — создать новый файл.
  • Ctrl + O — открыть файл.
  • Ctrl + S — сохранить файл.
  • Ctrl + Shift + S — сохранить как.
  • Ctrl + Q — выход из программы.

Редактирование текста

  • Ctrl + Z — отмена действия.
  • Ctrl + Y — повтор действия.
  • Ctrl + C — копировать.
  • Ctrl + X — вырезать.
  • Ctrl + V — вставить.

Управление приложением

  • Ctrl + R — выполнить команду «Пуск».

Рекомендации по использованию

  • Регулярно сохраняйте изменения, чтобы избежать потери данных.
  • Перед открытием нового файла убедитесь, что текущие изменения сохранены.
  • Используйте команды «Назад» и «Вперёд» для аккуратного редактирования текста.
  • Применяйте горячие клавиши для ускорения работы.
