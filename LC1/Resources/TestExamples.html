Тестовый пример

На рисунках 2–7 приведены примеры тестового запуска разработанного синтаксического анализатора, проверяющего корректность объявления вещественной константы с инициализацией на языке Kotlin.

1.

const val pi: Double = 3.14;

2.

val pi: Double = 3.14;

3.

cons@t val pi: Double = 3.14;

4.

const: val pi: Double = 3.14;

5.

const val pi: Int = 3.14;

6.

const val pi: Double = 3.1@4;
