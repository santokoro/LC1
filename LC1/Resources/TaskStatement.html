Постановка задачи

Константы в языке Kotlin объявляются с ключевым словом const и используются для значений, известных на этапе компиляции. Для объявления именованной константы с явным типом и инициализацией применяется конструкция с ключевыми словами const и val, именем идентификатора, указанием типа после двоеточия и присваиванием значения.

Формат записи:

const val идентификатор : Double = вещественное_число ;

Примеры:

  1. Простая константа с положительным вещественным значением:
    const val pi: Double = 3.14;
  2. Константа с отрицательным вещественным значением:
    const val x: Double = -2.5;
  3. Другой идентификатор и другое число:
    const val rate: Double = 0.01;

В связи с разработанной автоматной грамматикой G и реализацией разбора на основе конечного автомата синтаксический анализатор (парсер) объявления вещественной константы будет считать верными строки, полностью удовлетворяющие указанному формату и правилам лексического разбора (набор допустимых лексем, границы ключевых слов, форма вещественного литерала и т.д.), например:

  1. const val pi: Double = 3.14;
  2. const val e: Double = 2.718;
