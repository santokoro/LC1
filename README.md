4. Вариант задания:
Вариант №54. Объявление вещественной константы с инициализацией на языке Kotlin.
В связи с разработанной автоматной грамматикой G[<Z>] синтаксический анализатор (парсер) создания функции будет считать верными следующие записи функций:
1)	“const val pi: Double = 3.14159;”
2)	“const val x:Double = 12.0;”
3)	“const val pi: Double = 3.14;”

5. Разработанная граматика: 
G[<Z>]:
1) <START> -> 'const' <const> 
2) <const> -> ' ' <space>
3) <space> -> 'val' <value>
4) <value> -> ' ' <spaceafter>
5) <spaceafter> -> letter <id>
6) <id> -> letter <id> | digit <id> | '_' <id> | ':'<after_colon>
8) <after_colon> -> 'Double' <type> 
9) <type> -> '=' <equal>
10)<equal> = '-' <int> | digit <intrem> 
11)<int> -> digit <intrem>
12)<intrem> -> digit <intrem> | '.' <double> 
13) <double> -> digit <before>
14)<before> -> digit <before> |';'
15)<digit> -> '0'|'...'|'9'

6. Классификация по Холмскому:
Данная грамматика является автоматной, так как соблюдается на ограничение формы записи правил:
A --> aB | a | e,

В разработанном синтаксическом анализаторе реализована расширенная система обработки ошибок, включающая:
1. Диагностику ошибок (обнаружение и формирование сообщений);
