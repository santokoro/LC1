8. Диагностика и нейтрализация ошибок:
Оновная идея метода Айронса такова: когда в процессе разбора встречается ошибка (то есть очередной символ входной цепочки не совпадает ни с одним из допустимых в текущей ситуации символов), структура входной последовательности имеет вид Tt, где T — это ошибочный символ, а t — все остальные символы после него.

Процедура исправления включает следующие этапы:

1. Выявляются все незавершённые фрагменты (недостроенные кусты) дерева разбора.

2. Строится множество L, состоящее из остаточных символов этих незавершённых фрагментов.

3. Из входной строки последовательно удаляются символы (начиная с ошибочного), пока строка не примет вид Tt, удовлетворяющий условию U ⇒ T, где U — некоторый символ из множества L. Иными словами, удаление продолжается до тех пор, пока следующий символ T не окажется выводимым из какого-либо остаточного символа незавершённых кустов.

4. Устанавливается, какой именно незавершённый фрагмент породил символ U (то есть, частью какого куста является U). Тем самым определяется, к какому месту дерева разбора можно «присоединить» оставшуюся после удаления ошибочного фрагмента входную цепочку.
